2.9 · Referral structure
  • →2-layer structure. Layer 1 earns % of referred user fees. Layer 2 earns % of L1 referrals' fees.
  • →Referral tier scales by number of qualified referrals. Qualified = referred user trades > $10.
  • →Access key generated when weekly trade volume > $10 — 1 key per user per week.
2.10 · Inactivity penalty
Balance < $5 continuously for 3 days → reset everything: Tier, Points, Leaderboard rank, Referral benefits.
  • →Passive warning (day 2): in-app banner if balance still < $5: "Your rewards reset in 24h if balance stays below $5."
  • →Active warning: when user initiates withdraw that would bring balance below $5 → show confirmation modal first.
